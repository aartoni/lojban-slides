    What if we ask a vacuous questions? Like ``Are all the elefant in this room pink?''

    \pause

    \begin{center}
        % na'i is the metalinguistic negator
        \textbf{go\textquotesingle i na\textquotesingle i}
    \end{center}

    \pause
    That's because:
    \[
        \forall x, (E(x) \wedge R(x)) \rightarrow P(x)
    \]
\end{frame}
